# PocketCloud

A file upload and sharing service with expiration times, built with Express, TypeORM, and AWS S3.

## Installation

To install dependencies:

```bash
bun install
```

## Running the Server

To run:

```bash
bun run index.ts
```

The server will start on port 3002 (or the port specified in your `.env` file).

## API Documentation

For complete API documentation, see [API.md](./API.md)
